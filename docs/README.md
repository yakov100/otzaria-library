# תיעוד — otzaria-library

מסמכים טכניים על מבנה המאגר ואופן יצירתו.

## מסמכים

- **[קישורים וכותרות — מבנה ויצירה](קישורים-וכותרות.md)** — איך נוצרים ארבעת סוגי ה"קישורים" ב־`seforim.db`: קישורים רגילים, קישורים־למילה, קישורי־טווח, ו"כותרות"/עליות. כולל סכימת ה־DB, נתיבי היצירה בשני הגנרטורים (Sefaria ואוצריא), פורמט הקלט, ומדריך "איך ליצור".

  **שורה תחתונה — מה תומך הגנרטור של אוצריא:**

  | מנגנון | אוצריא |
  |---|---|
  | קישורים רגילים | ✅ נתמך |
  | קישורים למילה | ✅ נתמך (צד־מקור, כשיש `start`/`end`) |
  | קישורי טווח | ✅ נתמך (`line_index_1_end`/`line_index_2_end` ב־`_links.json`) |
  | כותרות / עליות | ✅ נתמך (`alt_toc/<ספר>_alt_toc.json`) |
